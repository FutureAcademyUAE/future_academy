Security reminder before connecting: Make sure your Google Cloud OAuth Consent Screen is set to "Testing" (never publish), only your email is listed as a Test User, and your GitHub repo is set to Private. See the Security Guide for full instructions.

2. Google Cloud → OAuth Consent Screen → Status must be "Testing" — NEVER click "Publish App"
3. Test Users → keep ONLY future.academy.ae@gmail.com — no other email
4. Credentials → Edit Client ID → Authorized Origins → ONLY your GitHub Pages URL (remove localhost)
5. Enable 2-Step Verification on your Google account at myaccount.google.com/security
6. Paste your Client ID below and connect — the weekly re-login is a security feature, keep it
